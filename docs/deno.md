# Deno

## Formatting

In your `deno.json`, you can set some
[options](https://docs.deno.com/runtime/fundamentals/linting_and_formatting/#available-options)
for its `fmt` tool, e.g.

```json
{
  "fmt": {
    "singleQuote": true
  }
}
```
